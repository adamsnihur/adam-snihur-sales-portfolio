# Przepływy użytkownika

## 1. Szybka kwalifikacja

Wejście → proposition „Sprzedaż B2B oparta na relacji i analizie” → trzy obszary dowodu: Orange, S’ouvre, e-commerce → CTA „Pobierz CV”.

## 2. Ocena doświadczenia

Hero → doświadczenie sprzedażowe → sposób pracy → doświadczenie wspierające → CTA kontaktowe.

## 3. Kontakt

CTA „Porozmawiajmy” → klient pocztowy z gotowym tematem lub LinkedIn → dalsza rozmowa poza stroną.

## 4. Pobranie CV

CTA „Pobierz CV” → lokalny plik PDF → otwarcie lub pobranie w przeglądarce.

## Stany

- Domyślny: wszystkie treści są dostępne bez JavaScriptu.
- Ładowanie: brak blokujących zależności i loaderów.
- Błąd: CTA e-mail i LinkedIn pozostają niezależne od pobrania PDF.
- Ograniczony ruch: animacje wyłączone przez `prefers-reduced-motion`.

## Rozstrzygnięte pytania

- Język: polski.
- Backend i formularz: niepotrzebne.
- Podstawowa publikacja: osobny projekt GitHub Pages.
- Telefon: niepublikowany; kontakt przez e-mail i LinkedIn.
